3. Qmix P / Q+ / Q- / TC and V Modules

3.1. Edit device parameters - Overview

To edit the device paramaters of the above mentioned Qmix devices you can use the QmixElements CANopenTools Plugin.

  1. To open the CANopenTools Plugin, start the QmixElements software and select from the main menu the menu item Device -> Open Configuration.

  2. Then select the canopentools (shared) configuration:

  3. Connect your Qmix module to you BASE module and turn it on. Then click the Connect ❶ button in the QmixElements software and scan for connected devices ❷.

  4. The software should detect the connected module ❶. Click with the right mouse button on the entry with the detected device and select Assign EDS File ❷ (see figure below).

  5. Select the valid EDS file ChipF40.eds for the Qmix module from the existing EDS-files.

  6. You can now access all device and configuration parameters of the device:

Attention

Changing device parameters can cause malfunctions or cancel safety mechanisms. Only change device parameters as instructed by the technical support staff.
